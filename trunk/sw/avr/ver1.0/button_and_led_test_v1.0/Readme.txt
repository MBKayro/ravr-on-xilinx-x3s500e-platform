Note:
The code is implemented on Xilinx SPARTAN-3E demo board, complied by AVR Studio 5.0 (avr-gcc).
The code is fitted for  AVR core version 1.0(../../../../rtl/verilog/ver1.0). 
Function:
(1) Blink 1 LED at a one-second period when no button is pressed.
(2) Right shift blinking LED when pressing east button.
(3) Left shift blinking LED when pressing west button.
